Planes de desarrollo web para restaurantes

Tres niveles de servicio para que elijas cuánto quieres delegar.

Todas las opciones incluyen una web moderna, adaptada a móvil y orientada a reservas. Cambia únicamente el nivel de soporte, las revisiones y el tipo de acompañamiento.

Plan Básico
Presencia online

200 € desarrollo · 10 €/mes mantenimiento

Para estar en Internet de forma profesional, sin complicaciones.

  • Web moderna y responsive (móvil, tablet y ordenador).
  • Secciones: Inicio, Carta, Contacto, Reseñas.
  • SEO básico (estructura, títulos y descripciones).
  • Hosting y dominio incluidos.
  • Certificado SSL (https).
  • Mantenimiento técnico mensual.
  • Copias de seguridad periódicas.
  • Monitorización básica (web online).
Plan Pro
Recomendado

200 € desarrollo · 20 €/mes mantenimiento

Para tener la web siempre al día y con soporte rápido.

  • Todo lo del Plan Básico.
  • Soporte técnico con respuesta < 48 h.
  • Revisión trimestral de la web.
  • Pequeñas modificaciones cada trimestre (carta, textos, fotos).
  • Optimización de rendimiento trimestral.
  • Revisión de formularios y sistema de reservas.
Plan Premium
Tranquilidad total

200 € desarrollo · 40 €/mes mantenimiento

Para olvidarte de la parte técnica y hacer evolucionar la web.

  • Todo lo del Plan Pro.
  • Soporte prioritario.
  • Modificaciones y ajustes bajo demanda.
  • Revisión continua de rendimiento.
  • Auditorías periódicas de ciberseguridad.
  • Monitorización avanzada.
  • Asesoramiento técnico estratégico.
